My Projects

Here are some of the projects I've worked on:

Floryne (E-Commerence Website) GitHub

Floryne is a simple e-commerce website I developed during my 10th grade as part of an assessment project for class promotion. This application was built using native PHP and Bootstrap 4, without relying on any frameworks. It includes features such as product listings, a shopping cart, and basic order processing. Through this project, I gained a deeper understanding of web development fundamentals and how core web technologies work behind the scenes.

INVAS- Inventaris SMK Assalaam (Inventory Website) GitHub

INVAS (Inventaris SMK Assalaam) is a website developed as part of the P5 (Project to Strengthen the Profile of Pancasila Students) initiative and an assessment project for 11th-grade students. This system is designed to digitally manage inventory data within SMK Assalaam efficiently. Built using the Laravel framework, INVAS offers a robust, secure, and scalable structure. With this system, users can easily record, manage, and monitor incoming and outgoing items, as well as track current stock levels.

Matrix Multiplication (Mini Project Website) GitHub

This is a mini project I created in 11th grade—a 3x3 matrix multiplication web application built using JavaScript. I developed this project to deepen my understanding of logic and how mathematical operations can be implemented in programming. The website allows users to input two 3x3 matrices and instantly displays the result. Through this project, I improved my skills in JavaScript, array handling, and problem-solving in a web environment.
